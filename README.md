# Inside the System

**Inside the System** is an auditable process simulation / guided control-board RPG.

Core thesis:

> Business processes can be converted into playable, testable, auditable, and teachable decision environments.

## Current Status

- `index.html` contains **Inside the System v0.8 — Guided Control Board Prototype**.
- `archive/v0.7-proof-slice/` contains the frozen maintenance RPG proof slice.
- `docs/` contains the design lock, proof-slice lock, guardrails, parking lot, and v0.8 handoff.
